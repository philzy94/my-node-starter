# 📝 Node.js Blog API Starter Kit

This is a modular Express + TypeORM backend inspired by NestJS structure. It supports:

✅ JWT Auth (Login & Register)  
✅ Users & Posts CRUD  
✅ Swagger Docs  
✅ Logger Middleware  
✅ Docker support  
✅ Class-validator + class-transformer  
✅ Dependency Injection (tsyringe)

---

## 🛠️ Tech Stack

- Node.js + Express
- TypeScript
- PostgreSQL + TypeORM
- tsyringe (for DI)
- Swagger
- Docker
- Winston logger
